# Abstract Factory
| Category   | Difficulty | Frequency of use |
| ---------- | ---------- | ---------------- |
| Creational | Easy       | High             |

## Definition
Provide an interface for creating families of related or dependent objects without specifying their concrete classes.

## Structure
- `AbstractFactory`
- `ConcreteFactory`
- `AbstractProduct`
- `ConcreteProduct`
- `Client`
